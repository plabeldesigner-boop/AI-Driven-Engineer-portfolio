# Portfolio — Static Starter

This is a minimal, clean static homepage for a portfolio (HTML/CSS/JS). It contains:

- `index.html` — main page
 - `css/styles.css` — basic styling (responsive)
 - `js/script.js` — mobile nav + contact form mailto handler
- `about/index.html` — dedicated about page
- `about/css/about.css` — about page styling
- `about/js/about.js` — about page mobile nav script

To preview locally:

```bash
# from the project folder
python -m http.server 8000
# then open http://localhost:8000 in your browser
```

You can also open `index.html` directly in the browser.

Next steps:
